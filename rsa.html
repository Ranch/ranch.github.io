Rsa Project

Our program allows for decryption and encryption with the help of a users public key and the hosts private key. The keys get input when you choose what you want to do after the program gives you 3 choices. When you encrypt, the computer uses the public key to encrypt a message. After that the message is printed on the screen and is ready to be sent out. When you want to decrypt, it uses the private key in order to untangle the unwrap the message. After that you get your massage, happy messaging!

There were some difficulties in the process of making our program. For example we had an issue with a loop for the beginning of the program, you are given the choices to encrypt, decrypt, or stop the program. Whatever you chose, the program would stop no matter what, upon further inspection, we noticed that the loop we created was being stopped by any response so we had to rework the loop in order for the program to go any further. Our next problem was that the values used for encryption and decryption were being addressed before we even used them, we had to end up changing the code, depending on what option you chose(encrypt or decrypt) it would then ask you for those values instead of having then assigned before you did anything, the problem was that the values would be assigned before you did anything and would not allow the program to run correctly.
